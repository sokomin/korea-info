■ 수정 사항

- 점핑 UI 에서 가이드 탭 클릭 시, 간헐적으로 비정상 텍스트 노출되며, 렉 현상이 발생하는 현상

- 퀘스트 UI에서 추천 퀘스트 클릭 시, 클라이언트가 종료되는 현상

- 퀘스트 UI에서 추천 퀘스트 클릭 시, 다른 퀘스트가 나타나는 현상

- 제련 실패 시 제련 옵션 텍스트가 비정상적으로 나타나는 현상

- 레이드 및 비밀 던전 입장 시 출석체크 팝업창이 나타나는 현상

- 변신소녀 ‘교란술’ 전직이 불가능한 현상

- 소환사 ‘돌진 명령’ 스킬이 정상적으로 적용되지 않는 현상

- 이계 강화 진행 시 ‘이계의 강화석’아이템이 차감되지 않는 것처럼 보이는 현상

- ‘이상한/기묘한 단지’ 아이템 사용 후 몬스터를 처치하여도 경험치 획득이 되지 않는 현상

- 디멘션 허브/토벌 콘텐츠의 종료 이후, [마을로 돌아가기] 기능 사용이 불가능한 현상
