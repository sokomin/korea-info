안녕하세요. 붉은보석 입니다.

 

좀더 원활한 로그인 기능 제공을 위해 SNS 간편 로그인 기능이 추가 됩니다.

기존에 보유하고 계신 SNS 계정으로 가입 및 로그인이 가능하게 됩니다.

 

 

[SNS 간편 로그인]

- 적용일: 2021년 5월 26일(수) 정기점검 이후

 

1) 로그인 방법

- 홈페이지 로그인 영역 하단이나 로그인 페이지로 이동 후 구글, 카카오톡, 네이버 계정 로그인을 통해 이용이 가능합니다.

 


 

 


 

 

2) 간편 로그인설명

- 간편 로그인 아이디는 가입 시 특수문자가 포함된 임의의 문자열로 생성됩니다.

 


 

- 기존 가입된 아이디를 포함하여 동일한 명의의 아이디 3개를 모두 생성하신 경우 추가로 생성할 수 없습니다.

- 현재 가입된 간편 로그인 아이디는 [아이디찾기]를 통해 본인확인 후 확인이 가능합니다.

- 간편 로그인을 통해 로그인 한 아이디는 반드시 홈페이지 [게임시작]을 통해서만 게임 이용이 가능하며, 

  설치된 게임 런처에서의 로그인 기능은 제공하지 않습니다.

- 게임 실행 시 ActiveX 가 동작하므로 간편 로그인 아이디는 Internet Explorer 브라우저를 통해 이용을 부탁 드립니다.

- 간편 로그인 아이디로 당근 및 아이템 선물 시에는 간편 로그인 아이디가 아닌,

  가입 후 생성된 특수문자가 포함된 아이디로 보내야 합니다.

- 간편 로그인 계정은 MOTP 가입을 진행 할 수 없으므로 2차 비밀번호를 설정해주시길 바랍니다.

 

※ 주의사항

- SNS 간편 로그인의 자동 로그인 해제를 원하실 경우 붉은보석 로그아웃과 함께 간편 로그인으로 가입한 

  플랫폼의 웹 브라우저 자동 로그인을 해제하셔야이후 자동 로그인이 진행되지 않습니다.

  (카카오 간편로그인 계정의 경우 붉은보석 로그아웃 후 브라우저 전체 종료)

- 보안을위해 다수가 이용하는 공공시설(ex. PC방, 학교 등)에서의 자동 로그인 기능은 가급적 지양해주시길 바랍니다.

 

언제나 더 나은 서비스를 제공 드리기 위해노력하겠습니다.

감사합니다.
