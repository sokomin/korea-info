7. 장비 강화 UI 개선

 

- 이계 강화 UI 에서 강화 대상 및 강화 보조제 아이템을 더블 클릭으로도 등록이 가능하도록 개선하였습니다.

- 이터널 강화, 유니크 연계 비법서 UI 에서 사용되는 아이템을 더블 클릭으로도 등록이 가능하도록 개선하였습니다.

- 이터널 강화 시 관련 재료를 제외한 다른 아이템의 경우 블러 처리가 되어 한눈에 볼 수 있도록 개선하였습니다.

- 반지 강화 UI 에서 더블 클릭으로 등록된 반지 아이템을 해제할 수 있도록 개선하였습니다.

- 반지 강화 UI 에서 강화 성공/실패 후 등록된 반지 아이템이 해제되지 않고 바로 강화가 가능하도록 개선하였습니다.

- 반지 강화 이팩트 지속시간을 이계 강화 이펙트 시간과 동일하게 개선하였습니다.

- 아이템 연마 UI 에서 아이템 등록 시 더블 클릭으로 등록이 가능하도록 개선하였습니다.

- 아이템 연마 UI 에서 연마 실패 후 등록된 연마 아이템이 해제되지 않고 바로 강화가 가능하도록 개선하였습니다

- 유물 아이템 강화 UI 에서 아이템 등록 시 더블 클릭으로 등록이 가능하도록 개선하였습니다.

- 유물 아이템 강화 UI 에서 강화 완료 후 등록된 유물 아이템이 해제되지 않고 바로 강화가 가능하도록 개선하였습니다.

- 유물 아이템 강화 이팩트 지속시간을 이계 강화 이펙트 시간과 동일하게 개선하였습니다.

- 모험가 협회에 장비 강화 안내를 담당하는 NPC 파운 (좌표 40, 55) 이 추가됩니다.

- NPC 파운은 장비 강화 관련 설명은 물론 각각의 장비 강화 NPC 위치를 안내하며, 원할 경우 해당 위치로도 이동 시켜 줍니다.

- 잠금된 코스튬은 아이콘 좌측 하단에 자물쇠 UI 가 표시됩니다.

- 잠금/해제 기능은 코스튬 우클릭을 통해 진행할 수 있습니다.

- 코스튬 분해 시 코스튬 강화 단계가 표시되도록 개선됩니다.

- 코스튬 분해 UI 에서 코스튬 이름의 색상이 등급별 색상과 동일하게 변경됩니다.

- 코스튬 분해 후 코스튬 분해 창이 바로 닫히지 않도록 개선됩니다.

- 코스튬 분해 UI 에서 분해에 필요한 재화가 체크되도록 개선됩니다.
